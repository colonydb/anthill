# Anthill

ColonyDB's design system.

## Getting Started

### Installation

PNPM:

```shell
pnpm add @colonydb/anthill
```

Yarn:

```shell
yarn add @colonydb/anthill
```

npm:

```shell
npm install @colonydb/anthill
```

### Usage

```jsx
import { Base } from '@colonydb/anthill/Base';
import { Button } from '@colonydb/anthill/Button';

const MyComponent = () => (
	<Base>
		<Button
			onClick={() => {
				alert('Hello!');
			}}
		>
			Click me
		</Button>
	</Base>
);
```
